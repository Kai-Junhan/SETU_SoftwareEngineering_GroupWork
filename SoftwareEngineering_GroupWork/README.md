# Library Management System - Version History







This directory contains three subfolders, each representing a revised version of the Library Management System. Below is a brief overview of each version:







## Version 1: Technical Framework



Lays the basic technical foundation of the system, including core package structure, initial class designs, and fundamental project architecture. Serves as the base for subsequent development.







## Version 2: Core Functions



Builds on Version 1 to implement essential business functionalities of the library management system. Enables basic system operation with core workflows.







## Version 3 ( Final Version ): Enhancements \\\& Improvements



Optimizes the system based on Version 2, with key additions including data storage functionality. Improves overall system robustness, usability, and performance.




（The data is stored in the TXT files located in the "src" directory at the same level as "NUIST-SETU-HW".）
